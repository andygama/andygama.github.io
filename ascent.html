Ascent

ASCENT is an acronym standing for Advanced Strategies for
Competing Effectively and Nurturing Technique.
This program is for archers who want to learn the technical aspects
of archery and build up knowledge and skill to compete in
archery tournaments. If you're brand new to archery and want to
learn all of its various aspects, this is the program for you!

Ascent is a great environment to learn archery,
with lots of one-on-one instruction as well as
interaction with senior archery members. All you
need to bring is yourself and a good attitude!

Ascent meets every Sunday morning, from 9 AM
to 12 PM. There is nor prior registration needed
to be part of Ascent, just show up!

DeCal
Through the DeCal Program on
the Berkeley campus, Cal
Archery is able to run a student
led archery class lead by other
students! This is a unique class
experience for any student
looking for an interesting in earning a P/NP unit. The class is taken through a set
curriculum and graded on attendance with no exams or grades! Anyone can
learn how to shoot and even build up to shooting competitively.

Admittance into the Archery DeCal is very competitive as there is a limited class size,
If you couldn't make it into the DeCal this semester and are still interested in learning
how to shoot, please consider trying out the Ascent Program!
